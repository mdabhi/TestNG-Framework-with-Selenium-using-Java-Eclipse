# This repository contains the project files for the live project completed for testing Banking application(Guru99).
## Technologies/ Tools used : Eclipse, Java, Selenium, Maven, TestNG framework, JUnit, POM 
- URL : https://www.demo.guru99.com/V4/
- Main Files: LoginTest.java, BasicUtils.java, testData.xlsx 
- Data file : testData.xlsx

## Accomplished:

- Created a Maven Project in Eclipse
- Setup TestNG plugin
- Edited Maven, TestNG, Selenium, JUnit, POI etc. dependencies in pom.xml
- Divided main in 3 methods, setUp, LoginPage, tearDown
- setUp method - browser configuration, getting BaseURL and opening the browser
- LoginPage method - Executing various testcases X no.of iterations as per Test data included in Data file
- tearDown method - closes browser once execution completed
